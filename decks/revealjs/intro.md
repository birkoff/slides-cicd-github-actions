## reveal.js The HTML Presentation Framework

Create, visualize and share fully-featured and beautiful presentations on any web browser.
---

### Anything you can do on the web, you can do in your presentation.
You can change styles with CSS, include external web pages using `<iframe>`, or add custom behavior using the JavaScript API.

---

## Key Features...

---

- **Markdown Support:** Write content using inline or external Markdown.

- **Code Syntax Highlighting:** Beautifully formatted code blocks courtesy of highlight.js.

- **Touch Optimized:** Smooth swiping on mobile phones and tablets.

- **Speaker View:** Includes a timer, preview of the upcoming slide, and speaker notes.
